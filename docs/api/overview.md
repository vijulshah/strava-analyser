## Authentication

All endpoints require Bearer token authentication via **OAuth 2.0**.

```http
Authorization: Bearer {access_token}
```

Key scopes:
| Scope | Access |
|-------|--------|
| `read` | Public data |
| `read_all` | Private data |
| `profile:read_all` | Full athlete profile |
| `profile:write` | Update profile |
| `activity:read` | Public activities |
| `activity:read_all` | All activities including private |
| `activity:write` | Create/update activities |
