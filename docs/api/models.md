## StreamSet

Keyed set of time-series streams. Each stream has the same number of data points.

| Key | Stream Type | Data | Description |
|-----|-------------|------|-------------|
| `time` | `TimeStream` | `integer[]` | Seconds elapsed from activity start |
| `distance` | `DistanceStream` | `float[]` | Distance from start (meters) |
| `latlng` | `LatLngStream` | `[float,float][]` | GPS coordinates [lat, lng] |
| `altitude` | `AltitudeStream` | `float[]` | Altitude above sea level (m) |
| `velocity_smooth` | `SmoothVelocityStream` | `float[]` | Smoothed speed (m/s) |
| `heartrate` | `HeartrateStream` | `integer[]` | Heart rate (bpm) |
| `cadence` | `CadenceStream` | `integer[]` | Cadence (rpm or steps/min) |
| `watts` | `PowerStream` | `integer[]` | Power output (watts) |
| `temp` | `TemperatureStream` | `integer[]` | Temperature (°C) |
| `moving` | `MovingStream` | `boolean[]` | Moving flag |
| `grade_smooth` | `SmoothGradeStream` | `float[]` | Smoothed gradient (%) |

Each stream object also contains:
- `series_type` — `"time"` or `"distance"` (x-axis basis)
- `original_size` — total data points before resolution thinning
- `resolution` — `"low"`, `"medium"`, or `"high"`
