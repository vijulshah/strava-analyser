## `GET` /segment_efforts

**Operation ID:** `getEffortsBySegmentId`  
**Summary:** List Segment Efforts  
**Required Scopes:** `activity:read`  

Returns a set of the authenticated athlete's segment efforts for a given segment. Requires `activity:read_all` for Only Me activities.

### Parameters

| Name | In | Required | Type | Description |
|------|----|----------|------|-------------|
| `segment_id` | query | ✅ | `integer` | The identifier of the segment |
| `start_date_local` | query | ❌ | `datetime` | ISO 8601 — filter efforts *after* this date (local time) |
| `end_date_local` | query | ❌ | `datetime` | ISO 8601 — filter efforts *before* this date (local time) |
| `per_page` | query | ❌ | `integer` | Number of items per page (default 30) |

### Date Filtering Notes

- `start_date_local` and `end_date_local` use ISO 8601 format, e.g. `2024-01-01T00:00:00Z`
- Default returns the most recent 200 efforts
- Combine with `per_page` to paginate results

### Response

Returns: **`DetailedSegmentEffort[]`**

Each effort includes:
- `id`, `name` (segment name)
- `elapsed_time`, `moving_time` in seconds
- `start_date`, `start_date_local`
- `distance` in meters
- `average_cadence`, `average_watts`, `average_heartrate`, `max_heartrate`
- `kom_rank` — KOM rank (null if not top 10)
- `pr_rank` — PR rank (null if not top 3)
- `is_kom` — KOM/QOM flag
- `hidden` — hidden from public segment page

### Example

```bash
# All efforts on segment 12345 in January 2024
$ http get "https://www.strava.com/api/v3/segment_efforts?segment_id=12345&start_date_local=2024-01-01T00:00:00Z&end_date_local=2024-01-31T23:59:59Z" \
  "Authorization: Bearer [[token]]"
```
