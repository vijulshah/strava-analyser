**Operation ID:** `getRouteAsGPX`  
**Summary:** Export Route GPX  
**Required Scopes:** `read`  

Returns the given route in GPX format. The authenticated athlete must own the route. The response is a GPX XML file.

### Parameters

| Name | In | Required | Type | Description |
|------|----|----------|------|-------------|
| `id` | path | ✅ | `long` | The identifier of the route |

### Response

### Example

```bash
$ http get "https://www.strava.com/api/v3/routes/{id}/export_gpx" \
  "Authorization: Bearer [[token]]" > route.gpx
```

## `GET` /athletes/{id}/routes

**Operation ID:** `getRoutesByAthleteId`  
**Summary:** List Athlete Routes  
**Required Scopes:** `read`

### Parameters

| Name | In | Required | Type | Description |
|------|----|----------|------|-------------|

| `page` | query | ❌ | `integer` | Page number (default 1) |
| `per_page` | query | ❌ | `integer` | Number of items per page (default 30) |

### Response

### Example
