## `GET` /athlete

**Operation ID:** `getLoggedInAthlete`  
**Summary:** Get Authenticated Athlete  
**Required Scopes:** `read`  

Returns the currently authenticated athlete. Tokens with `profile:read_all` scope will receive a detailed athlete representation; all others will receive a summary representation.

### Parameters

No parameters required.

### Response

Returns: **`DetailedAthlete`**

Key fields returned:
- `id`, `username`, `firstname`, `lastname`
- `city`, `state`, `country`
- `sex`, `premium` (Summit), `summit`
- `follower_count`, `friend_count`
- `measurement_preference` ‚Äî `feet` or `meters`
- `ftp` ‚Äî Functional Threshold Power (watts)
- `weight` ‚Äî in kg
- `profile`, `profile_medium` ‚Äî profile photo URLs
- `clubs[]`, `bikes[]`, `shoes[]`

### Example

```bash
$ http get "https://www.strava.com/api/v3/athlete" \
  "Authorization: Bearer [[token]]"
```
