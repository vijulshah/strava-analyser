# Strava API v3 – Streams

> Auto-generated by `scripts/scrape_strava_docs.py` from https://developers.strava.com/docs/reference

**Total endpoints:** 4

Streams are the time-series data associated with an activity. Each stream is a list of data points of a particular type. All streams have the same number of data points and the data points at the same index are associated with the same time.

## Available Stream Types

| Key | Data Type | Description |
|-----|-----------|-------------|
| `time` | `integer[]` | Elapsed time from start in seconds |
| `distance` | `float[]` | Distance from start in meters |
| `latlng` | `[float, float][]` | GPS coordinates [latitude, longitude] |
| `altitude` | `float[]` | Altitude above sea level in meters |
| `velocity_smooth` | `float[]` | Smoothed speed in m/s |
| `heartrate` | `integer[]` | Heart rate in bpm |
| `cadence` | `integer[]` | Cadence in rpm (cycling) or steps/min (running) |
| `watts` | `integer[]` | Power output in watts |
| `temp` | `integer[]` | Temperature in Celsius |
| `moving` | `boolean[]` | Whether the athlete was moving |
| `grade_smooth` | `float[]` | Smoothed gradient in percent |

## `GET` /activities/{id}/streams

**Operation ID:** `getActivityStreams`  
**Summary:** Get Activity Streams  
**Required Scopes:** `activity:read` (public), `activity:read_all` (private)  

Returns the given activity's streams. Request specific stream types by listing them in the `keys` parameter.

### Parameters

| Name | In | Required | Type | Description |
|------|----|----------|------|-------------|
| `id` | path | ✅ | `long` | The identifier of the activity |
| `keys` | query | ✅ | `string[]` | Comma-separated list of stream types (e.g. `time,heartrate,watts,latlng`) |
| `key_by_type` | query | ✅ | `boolean` | Must be `true` — returns a map keyed by stream type |

### Response

Returns: **`StreamSet`**

```json
{
  "time": {
    "data": [0, 1, 2, 3, ...],
    "series_type": "time",
    "original_size": 3600,
    "resolution": "high"
  },
  "heartrate": {
    "data": [142, 145, 148, 150, ...],
    "series_type": "time",
    "original_size": 3600,
    "resolution": "high"
  },
  "latlng": {
    "data": [[37.8, -122.4], [37.81, -122.41], ...],
    "series_type": "distance",
    "original_size": 3600,
    "resolution": "high"
  }
}
```

### Resolution Levels

| Resolution | Description |
|------------|-------------|
| `low` | ~100 data points |
| `medium` | ~1000 data points |
| `high` | All data points |

### Example

```bash
# Get time, HR, power, and GPS streams
$ http get "https://www.strava.com/api/v3/activities/{id}/streams?keys=time,heartrate,watts,latlng&key_by_type=true" \
  "Authorization: Bearer [[token]]"
```
