### Response

- `id`, `name`, `description`
- `profile`, `profile_medium` — club logo URLs
- `club_type`, `sport_type`
- `city`, `state`, `country`
- `private`, `member_count`, `featured`, `verified`

- `membership` — `"member"` / `"pending"` / null
