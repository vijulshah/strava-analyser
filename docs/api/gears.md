## `GET` /gear/{id}

**Operation ID:** `getGearById`  
**Summary:** Get Equipment  
**Required Scopes:** `read`  

Returns a specific gear (bike or pair of shoes) for the authenticated athlete. IDs for bikes have the prefix `b`, for shoes the prefix `g` (e.g. `b1234567`, `g9876543`).

### Parameters

| Name | In | Required | Type | Description |
|------|----|----------|------|-------------|
| `id` | path | ✅ | `string` | The identifier of the gear (e.g. `b12345` for bikes, `g12345` for shoes) |

### Response

Returns: **`DetailedGear`**

| Field | Type | Description |
|-------|------|-------------|
| `id` | `string` | Gear identifier |
| `primary` | `boolean` | Whether this is the primary gear |
| `name` | `string` | Gear name |
| `distance` | `float` | Recorded distance in meters |
| `brand_name` | `string` | Brand name |
| `model_name` | `string` | Model name |
| `frame_type` | `integer` | Frame type (bikes only): 1=MTB, 2=Cross, 3=Road, 4=Time Trial |
| `description` | `string` | Gear description |
| `nickname` | `string` | Short nickname |

### Example

```bash
# Get bike details
$ http get "https://www.strava.com/api/v3/gear/b12345678" \
  "Authorization: Bearer [[token]]"

# Get shoe details
$ http get "https://www.strava.com/api/v3/gear/g87654321" \
  "Authorization: Bearer [[token]]"
```

### Notes

- Gear IDs are returned in `DetailedAthlete.bikes[]` and `DetailedAthlete.shoes[]`
- Activities reference gear via the `gear_id` field in `DetailedActivity`
- `distance` accumulates across all activities that used this gear
