## `GET` /segments/explore

**Operation ID:** `exploreSegments`  
**Summary:** Explore Segments  
**Required Scopes:** `read`  

Returns up to 10 of the top segments within a given bounding box. Results can be filtered by activity type and climb category. The activity type is determined by the athlete's profile type.

### Parameters

| Name | In | Required | Type | Description |
|------|----|----------|------|-------------|
| `bounds` | query | ✅ | `float[]` | Bounding box corners lat/lon, as `SW_lat,SW_lng,NE_lat,NE_lng` |
| `activity_type` | query | ❌ | `string` | Filter by type: `running` or `riding` |
| `min_cat` | query | ❌ | `integer` | Minimum climb category: 0 (uncategorised) to 5 (HC) |
| `max_cat` | query | ❌ | `integer` | Maximum climb category: 0 to 5 |

### Climb Categories

| Value | Label | Description |
|-------|-------|-------------|
| 0 | NC | Not categorised |
| 1 | 4 | Cat 4 |
| 2 | 3 | Cat 3 |
| 3 | 2 | Cat 2 |
| 4 | 1 | Cat 1 |
| 5 | HC | Hors Catégorie |

### Response

Returns: **`ExplorerResponse`**

```json
{
  "segments": [
    {
      "id": 229781,
      "name": "Hawk Hill",
      "climb_category": 1,
      "climb_category_desc": "4",
      "avg_grade": 5.7,
      "start_latlng": [37.8331119, -122.4834356],
      "end_latlng": [37.8280722, -122.4981393],
      "elev_difference": 152.8,
      "distance": 2684.8,
      "points": "polyline_encoded_string",
      "starred": false
    }
  ]
}
```

### Example

```bash
# Find segments in San Francisco area
$ http get "https://www.strava.com/api/v3/segments/explore?bounds=37.7,-122.5,37.9,-122.3&activity_type=riding" \
  "Authorization: Bearer [[token]]"
```

## `GET` /segments/{id}

## `PUT` /segments/{id}/starred

**Operation ID:** `starSegment`  
**Summary:** Star Segment  
**Required Scopes:** `activity:write`  

Stars or un-stars a segment for the authenticated athlete. Returns the segment with `starred` field updated.

### Parameters

| Name | In | Required | Type | Description |
|------|----|----------|------|-------------|
| `id` | path | ✅ | `long` | The identifier of the segment |
| `starred` | body | ✅ | `boolean` | `true` to star, `false` to un-star |

### Response

Returns: **`DetailedSegment`** (with updated `starred` field)

### Example

```bash
# Star a segment
$ http put "https://www.strava.com/api/v3/segments/{id}/starred" \
  "Authorization: Bearer [[token]]" \
  starred:=true
```
